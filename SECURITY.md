# Security Policy

Shortless runs entirely in your browser, talks only to youtube.com and collects no data. Still, if you find a security issue (e.g. a way to make the extension fetch foreign URLs, injection via channel names/whitelist entries, or anything else that could harm users), please report it responsibly.

## Reporting a vulnerability

- **Email:** support@digitmedia.de
- Alternatively, use GitHub's [private vulnerability reporting](https://github.com/Lazybone/Shortless/security/advisories/new) if enabled.

Please do **not** open a public issue for security problems. You should receive a response within a few days. Supported version: the latest release.
